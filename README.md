# PayFlow.PointsApi

Loyalty points engine for the PayFlow e-commerce platform.

## How to run

```bash
cd PayFlow.PointsApi
dotnet run
```

## How to access Swagger

Once the app is running, open your browser at:

```
http://localhost:5000/swagger
```

## Endpoints

| Method | Path | Description |
|--------|------|-------------|
| GET | /health | Returns service health status |
| GET | /api/points/ping | Ping the Points API |
